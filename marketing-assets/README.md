# Harvest Drone Marketing Assets

Production-ready direct response assets for the two core Harvest Drone funnels:

- Growers
- Operators

## Files

- `grower-vsl.md`
- `operator-vsl.md`
- `grower-ads.md`
- `operator-ads.md`
- `meta-grower-ads.md`
- `meta-operator-ads.md`
- `grower-email-drip.md`
- `operator-email-drip.md`
- `resend-grower-emails.md`
- `resend-operator-emails.md`
- `sms-follow-ups.md`
- `landing-sections.md`

## Usage Notes

- Each file is written for a single audience and a single goal.
- The copy is structured to be easy to drop into landing pages, ad platforms, email tools, and SMS workflows.
- Personalization tokens such as `{{first_name}}` are kept simple for CRM or email platform merge fields.
