# RDO Enterprise Drone Division Demo Runbook

Use this as the live walkthrough beside `output/rdo-enterprise-drone-division-demo.pptx`.

## Opening Frame

Position Harvest as the operating partner for an internal drone application division.

Say:

> RDO owns the capability. Harvest provides the implementation playbook, OS, training workflow, support process, maintenance coordination, and readiness governance.

Avoid saying Harvest issues FAA, state, pesticide, or manufacturer certifications.

## Live Demo Path

1. Open `/enterprise/rdo/division`.
   - Show the command center: operator readiness, fleet readiness, jobs, support load.
   - Point out that this is division readiness, not a one-off spray quote.

2. Open `/enterprise/rdo/blueprint`.
   - Explain the buildout plan: equipment, training, SOPs, dispatch, records, and reporting.
   - Emphasize that Harvest is selling implementation and operating assistance.

3. Open `/enterprise/rdo/spray-calendar`.
   - Show repeated potato spray windows.
   - Explain why internal capability matters when weather windows are narrow.

4. Open `/enterprise/rdo/readiness`.
   - Select `GF Potato 18 rescue pass`.
   - Keep Ben Castillo and `RDO-HY-02` selected.
   - Show the blocker list.

5. Say the blocker readout clearly:
   - Operator practical signoff is incomplete.
   - North Dakota pesticide applicator credential is missing or not verified.
   - Aircraft calibration is overdue.
   - Battery inspection checklist is incomplete.
   - SOP checklist and job documents are incomplete.

6. Open `/enterprise/rdo/operators`.
   - Show mixed readiness across the operator bench.
   - Click an operator training profile if the audience asks how readiness is sourced.

7. Open `/enterprise/rdo/fleet`.
   - Show ready and blocked aircraft states.
   - Tie maintenance blockers to dispatch decisions.

8. Open `/enterprise/rdo/application-records` and `/enterprise/rdo/support`.
   - Show that the division keeps records and support history after the application.

9. Open `/enterprise/rdo/performance`.
   - Close with acres applied, ready jobs, utilization, and avoided outsource cost.

## Close

Say:

> The value is not just spraying acres. The value is helping RDO build a safe, auditable, internal drone application division that improves every season.
