# RDO Demo Readiness Checklist

## Required

- [x] RDO enterprise dashboard exists.
- [x] RDO demo organization/data exists.
- [x] Operator readiness view exists.
- [x] Credential status is visible through operator profiles and readiness blockers.
- [x] Fleet readiness view exists.
- [x] Spray calendar exists.
- [x] At least one job shows readiness blockers.
- [x] At least one completed application record exists.
- [x] Support and maintenance ticket examples exist.
- [x] UI has been checked at desktop and mobile widths for core RDO routes.
- [x] Demo fixture is labeled as not live RDO production data.
- [x] RDO demo script exists.
- [x] RDO seed export exists for pilot data review.
- [x] Pilot-style local add/manage workflows exist for operators, aircraft, jobs, records, and support tickets.
