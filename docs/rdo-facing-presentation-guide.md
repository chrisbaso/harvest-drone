# RDO-Facing Presentation Guide

Use `output/rdo-facing-enterprise-drone-division-proposal.pptx` for an external RDO conversation.

This deck is intentionally different from the internal Harvest demo deck. It speaks to RDO as the owner of the capability, with Harvest as the implementation and operating-system partner.

## Talk Track

1. **Open with ownership**
   - "This is not a custom applicator pitch. The model is that RDO owns the internal drone application capability."
   - "Harvest helps stand it up with the OS, training workflow, readiness gates, support process, and operating rhythm."

2. **Tie it to RDO's public posture**
   - RDO publicly emphasizes responsible, science-driven potato farming and a field-by-field operating model.
   - The deck frames drone application as another governed operating capability, not a gadget.

3. **Name the practical problem**
   - "When a spray window opens, the question is not whether drones are interesting. The question is whether this operator, aircraft, job documentation, and field assignment can safely go today."

4. **Show the operating model**
   - RDO owns aircraft, operators, field priorities, and crop decisions.
   - Harvest provides implementation, workflow, readiness logic, support process, and reporting structure.

5. **Move into the live product**
   - Open `/enterprise/rdo/division`.
   - Explain that this is the command center RDO would use to manage its internal division.

6. **Make readiness the live moment**
   - Open `/enterprise/rdo/readiness`.
   - Select the blocked job.
   - Read the blockers as operational facts, not software alerts.

7. **Close with the workshop**
   - "The next step is a readiness design workshop where we map RDO's first crop/location scope, operator roster, aircraft roster, credential matrix, SOP matrix, and pilot spray calendar."

## Language To Use

- RDO-owned capability
- Harvest-enabled operations
- credential tracked
- credential verified
- Harvest-qualified
- ready under Harvest rules
- configurable by state, aircraft, payload, and operation type

## Language To Avoid

- Harvest certifies operators
- Harvest issues FAA certification
- Harvest issues pesticide applicator licenses
- Harvest replaces official Hylio training
