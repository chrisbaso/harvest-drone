# RDO Stack Roadmap

The operating-stack work is meant to support, not replace, the larger RDO and enterprise drone-division roadmap.

## Now

- Use Jobber for clients, requests, quotes, jobs, scheduling, and invoices.
- Use Jobber's QuickBooks Online sync for accounting.
- Use Twilio for SMS and Slack for internal alerts.
- Use Harvest OS to log events, map external IDs, and surface readiness context.

## Next

- Attach Jobber jobs to Harvest OS application jobs.
- Use Jobber status changes to trigger drone-readiness checks.
- Route customer SMS replies to the right internal Slack channel.
- Add Slack alerts for open support tickets, fleet blockers, and RDO readiness blockers.

## Enterprise Layer

Harvest OS should continue building:

- enterprise drone division dashboard
- RDO workspace
- operator training
- credential readiness
- fleet readiness
- spray calendar
- application records
- support tickets
- performance reporting

## Later

Add direct QuickBooks analytics only when Harvest Drone needs financial detail that Jobber cannot provide, such as margin by aircraft, territory, enterprise account, operator, or application program.
